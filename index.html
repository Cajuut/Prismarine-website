主な機能

サーバー管理に必要なすべてを、ひとつのアプリで。

🖥️

マルチサーバー管理

複数のMinecraftサーバーを同時に作成・起動・停止。それぞれ独立した設定で管理できます。

🔌

自動ポート開放

UPnPによる自動ポートフォワーディング。ルーター設定不要で外部からの接続を許可。

📊

リアルタイム監視

CPU・メモリ使用率、プレイヤー数をリアルタイムで監視。サーバーログも即座に確認。

🔄

自動再起動

指定間隔または指定時刻での自動再起動機能。サーバーの安定稼働をサポート。

🛡️

Firewall連携

Windows Firewallと自動連携。必要なポートを自動的に開放します。

🎮

クロスプレイ対応

Geyser/Floodgateをワンクリックでインストール。統合版からの接続を簡単に許可。

対応サーバータイプ

主要なMinecraftサーバーソフトウェアをすべてサポート。

バニラ / プラグイン
  • Vanilla - 公式サーバー
  • Paper - 最適化 (推奨)
  • Purpur - さらなる最適化
  • Spigot - 自動ビルド対応
MOD / ハイブリッド
  • Forge - MOD対応
  • Fabric - 軽量MODローダー
  • Mohist - Forge + Bukkit
  • Banner - Fabric + Bukkit
プロキシサーバー
  • Velocity - 高速プロキシ (推奨)
  • Waterfall - BungeeCord改良版
  • BungeeCord - 定番プロキシ
